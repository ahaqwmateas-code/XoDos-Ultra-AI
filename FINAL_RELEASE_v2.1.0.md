### 🚀 **How to Install**

1. **Download the APK:**
   ```bash
   wget https://github.com/ahaqwmateas-code/XoDos-Ultra-AI/releases/download/v2.1.0/xodos-ultra-ai-v2.1.0.apk
   ```

2. **Install on Android:**
   ```bash
   adb install xodos-ultra-ai-v2.1.0.apk
   ```

3. **Or install manually:**
   - Download the APK from the GitHub release page
   - Transfer to your Android device
   - Open with a file manager and tap to install
   - Grant necessary permissions when prompted

### 📝 **Build Instructions**

```bash
# Clone the repository
git clone https://github.com/ahaqwmateas-code/XoDos-Ultra-AI.git
cd XoDos-Ultra-AI

# Install dependencies
flutter pub get

# Build the APK
flutter build apk --release

# Build for Termux
flutter build apk --release --target-platform android-arm64
```
